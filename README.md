# analisis-deforestasi-dan-sawit-2024

## Analisis Hubungan Luas Lahan Kelapa Sawit dan Produksi Sawit terhadap Angka Deforestasi di Indonesia Tahun 2024

Repository ini berisi hasil proyek Data Wrangling dan Exploratory Data Analysis (EDA) yang bertujuan untuk menganalisis hubungan antara luas lahan kelapa sawit, produksi kelapa sawit, dan angka deforestasi di Indonesia pada tahun 2024. Proyek ini mencakup seluruh tahapan data wrangling, mulai dari pengumpulan data, pembersihan data, transformasi, hingga penggabungan beberapa sumber data yang relevan. Setelah data siap, dilakukan analisis eksploratif untuk mengidentifikasi pola, tren, serta potensi hubungan antar variabel yang berkaitan dengan isu lingkungan, khususnya deforestasi.

### Tujuan Proyek
- Mengkaji hubungan antara perluasan lahan kelapa sawit dan tingkat deforestasi di Indonesia.
- Menganalisis keterkaitan produksi kelapa sawit terhadap perubahan tutupan hutan.
- Menyajikan insight berbasis data sebagai dasar pemahaman dampak lingkungan dari sektor perkebunan kelapa sawit.

### Metodologi
- Data Wrangling (Cleaning, Transforming, Integrating)
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Visualisasi Data

### Output
- Dataset hasil pembersihan dan pengolahan data
- Visualisasi hubungan antar variabel
- Insight awal terkait hubungan antara kelapa sawit dan deforestasi

Repository ini diharapkan dapat menjadi referensi pembelajaran dalam penerapan data wrangling serta memberikan gambaran berbasis data mengenai isu deforestasi di Indonesia.
